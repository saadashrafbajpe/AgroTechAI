# 🌱 AgroTech AI – Edge-Based Crop Monitoring

AgroTech AI is an **offline, edge-based crop monitoring and decision support system** built using **Flask** and **SQLite**. It helps farmers access crop information, disease and pest management, weather advisories, growth monitoring, and farming recommendations without requiring an internet connection.

## ✨ Features
- 🌾 Crop Information
- 🐛 Pest Management
- 🍂 Disease Information
- 🌦 Weather Advisory
- 📈 Growth Monitoring
- 💡 Farming Recommendations
- 💾 Offline SQLite Database

## 🛠 Tech Stack
- Python
- Flask
- SQLite
- HTML, CSS, JavaScript

## 🚀 Run Locally

```bash
git clone https://github.com/yourusername/agrotech-ai.git
cd agrotech-ai
pip install -r requirements.txt
python app.py
```

Open your browser at:

```
http://127.0.0.1:5000
```
