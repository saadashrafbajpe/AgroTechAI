# 🌱 AgroTech AI

**Offline, Edge-Based Crop Monitoring & Decision Support System**

AgroTech AI is an offline crop monitoring system built with **Flask** and **SQLite**. It provides farmers with crop information, disease and pest management, weather advisories, growth monitoring, and farming recommendations without requiring an internet connection.

## 🚀 Run the Project

Open a terminal in the project folder and run:

```bash
python app.py
```

Then open your browser and visit:

```
http://127.0.0.1:5000
```

## ✨ Features

* 🌾 Crop Information
* 🐛 Pest Management
* 🍂 Disease Information
* 🌦 Weather Advisory
* 📈 Growth Monitoring
* 💡 Farming Recommendations
* 💾 Offline SQLite Database

## 🛠 Tech Stack

* Python
* Flask
* SQLite
* HTML, CSS, JavaScript

## 🌿 Supported Crops

* Coconut
* Arecanut
* Rice
* Rubber
* Cashew
